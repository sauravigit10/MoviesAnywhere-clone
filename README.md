🎬 MovieAnywhere Clone

    A MovieAnywhere-inspired web app built with React, allowing users to browse movies and TV shows, search dynamically, and watch trailers.
    This project showcases Netflix-style UI patterns including horizontal scroll rows, search overlays, and responsive layouts.

🌟 Features

    Home Page
    
     -> Browse movies by category (Popular, Trending, etc.)
    
     -> Netflix-style horizontal scrolling rows
    
     -> Hover effects on posters
    
    See All Page
    
    -> Display all movies in a category as a grid
    
    -> Back arrow navigation
    
    Movie Search
    
     -> Live search filtering with overlay
    
     -> Quick access to titles
    
    Movie Trailers
    
     -> Click on posters to open YouTube trailers via TMDb API
    
    Responsive UI
    
    -> Mobile and desktop friendly
    
    -> Dynamic grid wrap for “See All” pages.

🛠 Tech Stack

    -> React – Frontend library
    
    -> React Router DOM – Navigation
    
    -> React Icons – FontAwesome icons
    
    -> TMDb API – Movie/TV data & trailers
    
    -> CSS Flexbox & Grid – Layouts & scrolling
    
    -> JavaScript Fetch API – Data fetching
